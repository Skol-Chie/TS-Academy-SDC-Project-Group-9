# TS Academy Planets Explorer – Capstone Project Group 9

## Project Overview

This project is a **Software Engineering Capstone Project** built with **React** as part of the TS Academy curriculum. The goal is to recreate a provided Figma mockup into a fully functional, responsive web application using modern frontend tools and **Git collaboration workflows**.

The application fetches dynamic data about planets using the Fetch API and presents it in a clean, component-based interface. It also includes smooth scrolling CTAs, a functional contact form with validation, and responsive layouts for mobile and desktop.

## Application Features

* "Explore the Data" CTA scrolls to planets section
* "Contact Us" CTA scrolls to the form section
* Planet data fetched dynamically:

  * Name
  * Distance from the sun
  * Image (wrapped in `<figure>` element)
* Video element (autoplay, muted, loop – no iframe)
* Responsive design (mobile & desktop)
* Contact form with client-side validation and submission feedback
* `<meta name="author">` contains the **group name**

## Git Workflow Rules (VERY IMPORTANT)

* `main` branch is **protected**
* Each task = **new branch**
* Naming convention:

  ```
  feature/hero-section
  feature/planets-fetch
  fix/navbar-scroll
  ```
* No one merges **their own PR**
* At least **1 review required** before merge
* Avoid unnecessary commits
